> Editable Concept Bottleneck Models
>
> Lijie Hu\* , Chenyang Ren\*, Zhengyu Hu\*, Hongbin Lin, Cheng-Long Wang, Zhen Tan, Weimin Lyu, Jingfeng Zhang, Hui Xiong, and Di Wang.
